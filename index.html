How to Solve the Time Zone Problem of our website in Aws Server

There are several ways to solve this problem but here I will provide you with the three ways to do this:

TimeZone, April 9, 2018
  1. Using terminal of our EC2 instance.
  2. Using the script in our php file.
  3. Using the php.ini file.

Using terminal of our EC2 instance:

  1. You should have running instance in Aws.
  2. Access your server through ssh.
  3. Open your ssh terminal
  4. Check the Defualt TimeZone with following command:
    timedatectl
  5. Now set the TimeZone according to your choice using this command:
    timedatectl set-timezone EST

Using Script in our php file:

Use the below code in your php file:

<?php date_default_timezone_set("Asia/Kolkata"); ?>

Using the php.ini file:

  1. Find the php.ini file.
  2. Follow the direction likewise:
    cd > cd/etc > cd php5 > cd apache2 > ls > nano php.ini
  3. Find this line: [Date] ; Defines the default time zone used by the date functions ; http://php.net/date.timezone ;date.timezone =
  4. Edit timezone likewise: [Date] ; Defines the default time zone used by the date functions ; http://php.net/date.timezone date.timezone =Asia/Kolkata
  5. restart your server using this command:
    service apache2 restart
